# Title
Comment On Your Major

# Features
- Sign in, out and up
- Comment on the major
- like the comment
- reply to the specific comment
- Update user's details
- Popularity analysis
- View signed up users with details

# Frameworks/Libraries & Database
- Vue 3 (FE Framework)
- Node.js Express (Typescript) (BE Framework)
- Mysql2 (DB)

# Deployment
- Frontend (Netlify)
- Backend (Heroku)
- Database (clearDB)
